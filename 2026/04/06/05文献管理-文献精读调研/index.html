AI 辅助文献精读调研

AI 辅助文献精读调研

[!summary]
AI 辅助文献精读，核心不是“替你读完”，而是帮你完成 5 件事：

  • 快速筛选
  • 辅助理解
  • 结构化提炼
  • 多篇比较
  • 笔记沉淀

最实用的组合通常是：
找文献工具 + 单篇精读工具 + 多篇对比工具 + Zotero + 笔记系统

一、工具速览

场景 推荐工具 作用
找相关文献 ResearchRabbit / Connected Papers / Elicit 建立文献地图、扩展种子论文、初筛
单篇精读 SciSpace / Explainpaper / ChatPDF 解释难句、理解方法、定位信息
多篇比较 NotebookLM / Elicit 做对比、分组、归纳争议点
判断论文可信度 scite 看后续研究是支持、质疑还是仅提及
长期沉淀 Zotero + PDF Translate + Better BibTeX 管理 PDF、翻译、引用、回收笔记

二、怎么选工具

1. 找文献 / 建立地图

  • ResearchRabbit：适合看引用关系和研究脉络
  • Connected Papers：适合从种子论文快速扩展
  • Elicit：适合问题驱动检索和初步表格整理

2. 单篇精读 / 看不懂就问

  • SciSpace：最适合“边读边问”
  • Explainpaper：适合逐句解释
  • ChatPDF / Humata：适合快速问答和定位内容
  • Scholarcy：适合批量初读和论文卡片，不适合替代精读

3. 多篇比较 / 写综述

  • NotebookLM：适合把多篇论文放在一起比较
  • Elicit：适合抽取研究问题、方法、结论并做表格

4. 判断靠不靠谱

  • scite：非常关键，能把“读懂”升级成“会评价”

5. 长期管理

  • Zotero：文献工作流底座
  • PDF Translate：中英对照阅读更高效
  • Better BibTeX：写作和引用管理更顺

[!important]
所有 AI 精读结果，最后最好都回收到 Zotero 或 Obsidian / Notion。
否则后面很难复用。

三、最实用的 3 套工作流

方案 A：入门型

适合：刚开始读论文，英文阅读有压力

组合

  • Connected Papers / ResearchRabbit
  • SciSpace / Explainpaper
  • Zotero + PDF Translate
  • Obsidian / Notion

流程

  1. 找核心论文
  2. 存入 Zotero
  3. 用 PDF Translate 辅助理解
  4. 用 SciSpace / Explainpaper 问方法、术语、难句
  5. 手动写精读卡：问题 / 方法 / 结果 / 局限 / 自己评价

方案 B：综述型

适合：写综述、开题报告

组合

  • Elicit
  • ResearchRabbit
  • NotebookLM
  • scite
  • Zotero

流程

  1. 用 Elicit 按问题找论文
  2. 用 ResearchRabbit 看文献网络
  3. 把核心论文放进 NotebookLM 比较
  4. 用 scite 看关键论文是否被支持或质疑
  5. 回写到 Zotero / Obsidian

方案 C：深度精读型

适合：已经锁定核心论文，要真正读透

组合

  • Zotero
  • SciSpace / Humata
  • scite
  • NotebookLM

流程

  1. 选 1 篇核心论文
  2. 用 SciSpace 逐段问：研究问题、方法流程、实验设计、结果支撑链
  3. 用 scite 看后续影响
  4. 再用 NotebookLM 和相关论文做横向比较
  5. 输出自己的批判性笔记

[!tip]
AI 负责解释、提炼、对比；你负责判断、批评、形成观点。

四、最值得保存的提示词

单篇精读

  • 抽结构：提取研究问题、方法、数据、结果、结论、局限
  • 讲人话：把一段学术表达重写成通俗中文，并解释术语
  • 看论证链：作者如何从问题一步步推到结论
  • 做批判：从研究设计、数据、方法、外推性四个角度分析
  • 找创新点：指出相对已有研究的新意到底在哪里

多篇比较

  • 对比方法：比较研究问题、方法、数据、指标、局限，并生成表格
  • 找分歧：哪些结论一致，哪些地方冲突，原因可能是什么
  • 为综述服务：按研究问题或方法路线分组

五、AI 精读最容易踩的坑

1. 不能只看 AI 总结

方法部分、结果图表、局限性一定要回原文确认。

2. 不要让 AI 替你理解

不要只问“结论是什么”，还要问：

  • 证据是什么？
  • 图表如何支持结论？
  • 有没有别的解释？

3. 要防幻觉和过度概括

相对更稳的工具：

  • NotebookLM
  • SciSpace
  • ChatPDF / Humata
  • scite

4. 不要让笔记散落在各个平台

AI 负责处理，Zotero / Obsidian / Notion 负责沉淀。

六、最终建议

轻量版

  • Zotero
  • Zotero PDF Translate
  • SciSpace
  • NotebookLM

进阶版

  • Zotero
  • PDF Translate
  • ResearchRabbit
  • Elicit
  • SciSpace
  • scite
  • NotebookLM

[!abstract]
一句话总结：
用 ResearchRabbit / Elicit 找文献，
用 SciSpace / Explainpaper 精读单篇，
用 NotebookLM 做多篇比较，
用 scite 做可信度验证，
最后把结果沉淀回 Zotero。
