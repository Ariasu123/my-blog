# Tags Page Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the default Fluid tags cloud with stable colored tag cards that split slash-delimited composite tags into individually rendered labels.

**Architecture:** Keep Hexo + Fluid intact and implement the change through project-side extension points only. Put normalization and stable color generation in a small pure utility module, expose it to EJS through a Hexo helper, then override the tags page template and style it with tags-page-only CSS. Because composite-tag splitting is limited to the display layer in this iteration, each rendered tag card reuses a real existing tag archive path: prefer an exact raw tag archive when one exists, otherwise reuse the first contributing raw tag archive.

**Tech Stack:** Hexo 8, Fluid theme, EJS template override, project-side Hexo helper script, CSS, Node.js verification via `node -e`

- [ ] **Step 2: Create the pure tag utility module**

Create `scripts/lib/tag-page.js` with:

```js
'use strict';

function splitTagName(name) {
  return String(name || '')
    .split('/')
    .map((part) => part.trim())
    .filter(Boolean);
}

function slugifyTag(name) {
  return encodeURIComponent(String(name || '').trim()).replace(/%20/g, '-');
}

function hashString(value) {
  let hash = 0;

  for (const char of String(value || '')) {
    hash = (hash * 31 + char.codePointAt(0)) | 0;
  }

  return Math.abs(hash);
}

function buildTagPalette(name) {
  const hash = hashString(name);
  const hue = hash % 360;
  const saturation = 68 + (hash % 8);
  const lightness = 88 - (hash % 6);

  return {
    background: `hsl(${hue} ${saturation}% ${lightness}%)`,
    border: `hsl(${hue} ${Math.max(saturation - 14, 48)}% ${Math.max(lightness - 14, 58)}%)`,
    text: `hsl(${hue} 42% 24%)`
  };
}

function normalizeTags(rawTags) {
  const cards = [];
  const seen = new Map();

  for (const rawTag of rawTags || []) {
    const rawName = String(rawTag && rawTag.name || '').trim();

    if (!rawName) {
      continue;
    }

    const rawPath = `tags/${slugifyTag(rawName)}/`;

    for (const name of splitTagName(rawName)) {
      if (!seen.has(name)) {
        const card = {
          name,
          rawPath,
          direct: rawName === name
        };

        seen.set(name, card);
        cards.push(card);
        continue;
      }

      const existing = seen.get(name);

      if (!existing.direct && rawName === name) {
        existing.rawPath = rawPath;
        existing.direct = true;
      }
    }
  }

  return cards
    .map((card) => ({
      name: card.name,
      rawPath: card.rawPath,
      palette: buildTagPalette(card.name)
    }))
    .sort((left, right) => left.name.localeCompare(right.name, 'zh-CN'));
}

module.exports = {
  splitTagName,
  slugifyTag,
  buildTagPalette,
  normalizeTags
};
```

Create `scripts/tag-page-helpers.js` with:

```js
/* global hexo */

'use strict';

const { normalizeTags } = require('./lib/tag-page');

hexo.extend.helper.register('build_tag_cards', function(siteTags) {
  const rawTags = typeof siteTags.toArray === 'function'
    ? siteTags.toArray()
    : Array.from(siteTags || []);

  return normalizeTags(rawTags).map((card) => ({
    name: card.name,
    path: this.url_for(card.rawPath),
    style: [
      `--tag-bg: ${card.palette.background}`,
      `--tag-border: ${card.palette.border}`,
      `--tag-text: ${card.palette.text}`
    ].join('; ')
  }));
});
```
